These are demo materials for demonstrating INTER-Mediator's fundamental function.

PHP files should be located as same directory as 'develop-im' folder.
The demo detabase for MySQL with the distribution should be installed and working.
The file params.php is same as distributed one.

1. open demo1.html, show the source.

2. change the extension to php, and add following source in the header seciton
=================================
<?php
	require_once( '../develop-im/INTER-Mediator/INTER-Mediator.php' );
	InitializePage(
		array( array(	'records' 	=> '1', 'name' => 'person', 'key' 	=> 'id',),), 
		null,
		array( 'db-class' => 'MySQL', 'db' => 'test_db', )
	);
?>
</head>
<body onload="doAtTheStarting();" onbeforeunload="return doAtTheFinishing();">
=================================

3. open demo1.php and proof that shows the contents of database.

4. add following code in any body part.
=================================
<?php GenerateConsole(); ?>
